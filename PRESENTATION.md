# Aury - Handmade Commerce Platform

## Overview
Aury is a modern marketplace platform connecting artisans and buyers, specializing in handcrafted products and creative patterns. Built with Next.js, Firebase, and Stripe Connect.

---

## 🎯 Key Features

### For Buyers
- **Curated Marketplace** - Browse unique, handcrafted products from verified creators
- **Secure Checkout** - Stripe-powered payment processing with buyer protection
- **Product Discovery** - Advanced search and filtering to find exactly what you need
- **Shopping Cart** - Seamless cart management with persistent sessions
- **Order Tracking** - Real-time order status and history

### For Sellers
- **Stripe Connect Integration** - Automated payouts and earnings tracking
- **Seller Onboarding** - Streamlined account setup with verification
- **Product Management** - Easy listing creation with image uploads
- **Earnings Dashboard** - Real-time analytics and revenue insights
- **Order Management** - Track sales and fulfill orders efficiently

### Community Features
- **Social Feed** - Share projects, patterns, and inspiration
- **Follow System** - Connect with favorite creators
- **Interactive Posts** - Like, comment, and engage with the community
- **Pattern Marketplace** - Buy and sell crochet/craft patterns

---

## 💻 Technical Stack

- **Frontend**: Next.js 15, React, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes, Firebase Admin SDK
- **Database**: Firestore (NoSQL)
- **Authentication**: Firebase Auth
- **Payments**: Stripe Connect (Express Accounts)
- **File Storage**: Firebase Storage
- **Deployment**: Vercel-ready

---

## 🔑 Core Capabilities

### Payment Processing
- Platform takes 5% fee on all transactions
- Automatic fund transfers to seller accounts
- Webhook integration for real-time payment updates
- Secure PCI-compliant checkout

### User Roles
- **Buyers** - Browse and purchase products
- **Creators** - Share content, Create listings and manage sales

### Security & Compliance
- Firebase Authentication with secure sessions
- Stripe-verified seller accounts
- Data validation and sanitization
- HTTPS enforcement

## 📊 Business Model
